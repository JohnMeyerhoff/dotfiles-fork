# mackup

> Keep your application settings in sync (OS X/Linux)

[github.com/lra/mackup](https://github.com/lra/mackup)

## Setup

### 1. Install binary (brew)

```sh
brew install mackup
```

### 2. Setup dotfiles

```sh
mackup restore
```

☝️ Carefully follow the instructions line by line if there are any conflicts.
